##meta ##convention

---

## Minimalist PKM System

**Core Philosophy:**
1. Simple as possible
2. Flexible - no strict conventions to remember
3. Tags + links + search for organization

## Filename Format

`{timestamp}-{descriptive-name}.md`

**Examples:**
- `202601292119-Claudia Yeni Çalışma Tarzı Güncellemesi.md`
- `202601292145-Sem YZ Module Outline.md`
- `202601290800-Bugünkü Plan.md`

**No type prefixes** - flexibility over structure

## Header Format

```markdown
# Title
##type_tag ##type_tag2
#topic_tag1 #topic_tag2 #topic_tag3

###c_draft (IF Claude wrote this - MANDATORY for Claude)

---
Content...

Related: [[link]]
```

## Tag System

### Type Tags (##double-hash)

Use for categorizing note type:
- `##daily_plan` - Daily planning
- `##project_plan` - Project planning
- `##meeting_notes` - Meeting notes
- `##literature_note` - Literature/research notes
- `##lecture_prep` - Lecture preparation
- `##idea` - Ideas, insights
- `##reflection` - Reflections, journal
- `##draft` - Draft documents
- `##reference` - Reference materials

**Flexible:** Create new type tags as needed

### Topic Tags (#single-hash)

Use for subject matter:
- `#ai` `#ethics` `#trauma` `#cbt` `#workshop` etc.

### Claude Draft Tag (###triple-hash)

- `###c_draft` — **MANDATORY when Claude writes content**
- Marks Claude-generated drafts for user review/editing
- User can remove tag after reviewing/editing

## Search Examples

**By type:**
- All daily plans: `##daily_plan`
- All project plans: `##project_plan`
- All Claude drafts: `###c_draft`

**By topic:**
- AI-related: `#ai`
- Workshop content: `#workshop`

**Combined:**
- Claude drafts about AI: `###c_draft #ai`
- Daily plans from January 29: `202601290 ##daily_plan`

**By filename:**
- Today's notes: `"20260129"`
- Specific note: `[[202601292119-Claudia]]`

## Workflow

### User creating note:
1. `Cmd+N` → The Archive generates timestamp
2. Write title
3. Add tags: `##type` and `#topics`
4. `---` separator
5. Write content
6. Rename: `{timestamp}-{descriptive-name}`

### Claude creating note:
1. Create file with timestamp + descriptive name
2. Add title
3. Add `##type_tag` and `#topic_tags`
4. **MANDATORY:** Add `###c_draft`
5. `---` separator
6. Write content

### Shared workspace:
- User's thoughts → The Archive
- Claude's drafts → The Archive (with `###c_draft`)
- Synthesis happens in shared space
- User edits/builds on Claude drafts

## Use Cases

1. **Long-term knowledge** - Zettelkasten linking
2. **Quick capture** - Send to Claude as context
3. **Thinking out loud** - Document thought process
4. **Claude drafts** - Outlines, plans, synthesis from Claude
5. **Collaboration** - Shared workspace for human-AI synthesis
