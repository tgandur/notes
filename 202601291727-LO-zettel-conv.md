#meta #convention
status: #active

---

## Filename Format

`{timestamp}-{TYPE}-{slug}.md`

Örnek: `202601291727-LO-zettel-conv.md`

## Type Prefixes

| Type | Prefix | Örnek |
|------|--------|-------|
| Project | PR | `202601281804-PR-sem-yz` |
| Lore | LO | `202601291703-LO-ai-ethics` |
| Note | NO | `202601291545-NO-meeting` |
| Reference | RE | `202601291200-RE-smith2024` |
| Daily Plan | DA | `202601290800-DA-Plan` |
| Daily Reflection | DA | `202601291730-DA-Reflection` |
| Journal | JO | `202601291900-JO-doubt` |

## Header Format

```
{timestamp} {Title}
#topic-tags
status: #active | #paused | #done | #someday
type: #project | #lore | #note | #ref | #daily | #journal
---

{content}

Related: [[links]]
```

## Rules

- Timestamp: `YYYYMMDDHHmm` (The Archive otomatik atar)
- Slug: küçük harf, 2-4 kelime, boşluk → `-`
- Status ve type: tag olarak (searchable)
- Links: `[[timestamp-TYPE-slug]]`

## Search Examples

- Tüm projeler: `"-PR-"`
- Aktif projeler: `"-PR-" #active`
- Research projeleri: `"-PR-" #research`
- Bugünün daily'leri: `"20260129" AND "-DA-"`
- Belirli not: `[[202601281804-PR-sem-yz]]`

## Workflow

1. `Cmd+N` → The Archive timestamp atar
2. Başlık yaz
3. Tags + status + type
4. `---`
5. Rename: `{timestamp}-{TYPE}-{slug}`
